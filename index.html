Create Alpine App

Welcome to Create Alpine App. A simple starter application for your Alpine and TailwindCSS applications.

  • Start a local dev server by running npm run dev

  • Hot reloading is enabled by default

  • Modify this page inside of index.html

  • To build your app, run npm run build

Need more functionality? Easily add some back-end magic by porting this app over to a Tall stack application.
